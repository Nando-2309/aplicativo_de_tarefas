 aplicativo_de_tarefas

Um aplicativo de tarefas gamificado, ou seja, mostra o ranking de cada tarefa através de uma barrinha indicadora mostrando a evolução do nível de cada tarefa até o atingimento da maestria, além disso há um formulário simples que após preenchido cria uma nova tarefa.

## Demonstração

![Descrição do GIF](assets/animação_de_aplicativo.gif)


## Funcionalidades

Liste as principais funcionalidades do seu projeto.

*   Mostra o nível de cada tarefa
*  Gerencia seu nível através da barrinha indicadora e níveis de faixas de cores ( basicamente os níveis seguem as faixas do Karatê sendo a faixa preta a penúltima e  a faixa coral, a última para sua graduação de mestre naquela tarefa)
*   Cria novas tarefas após o preenchimento do formulário.Para criação da tarefa o usuário só deve preencher o nome da tarefa, o nível de dificuldade dela e colocar uma url mostrando a imagem da tarefa
